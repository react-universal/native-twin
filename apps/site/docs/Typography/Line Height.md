# Line Height

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class           | Native | Web |
| --------------- | ------ | --- |
| leading-{t}     | ✅     | ✅  |
| leading-{a}     | ✅     | ✅  |
| leading-none    | ❌     | ✅  |
| leading-tight   | ❌     | ✅  |
| leading-snug    | ❌     | ✅  |
| leading-normal  | ❌     | ✅  |
| leading-relaxed | ❌     | ✅  |
| leading-loose   | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/line-height) ._
