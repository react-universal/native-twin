# Top, Right, Bottom, Left

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class        | Native | Web |
| ------------ | ------ | --- |
| inset-{t}    | ✅     | ✅  |
| inset-{a}    | ✅     | ✅  |
| inset-x-{t}  | ✅     | ✅  |
| inset-x-{a}  | ✅     | ✅  |
| top-{t}      | ✅     |  ✅  |
| top-{a}      | ✅     |  ✅  |
| bottom-{t}   | ✅     |  ✅  |
| bottom-{a}   | ✅     |  ✅  |
| left-{t}     | ✅     |  ✅  |
| left-{a}     | ✅     |  ✅  |
| right-{t}    | ✅     |  ✅  |
| right-{a}    | ✅     |  ✅  |
| inset-auto   | ❌     |  ✅  |
| inset-x-auto | ❌     |  ✅  |
| inset-y-auto | ❌     |  ✅  |
| top-auto     | ❌     |  ✅  |
| bottom-auto  | ❌     |  ✅  |
| left-auto    | ❌     |  ✅  |
| right-auto   | ❌     |  ✅  |
