# Divide Width

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class        | Native | Web |
| ------------ | ------ | --- |
| divide-x-{t} | ✅     | ✅  |
| divide-x-{a} | ✅     | ✅  |
| divide-y-{t} | ✅     | ✅  |
| divide-y-{a} | ✅     | ✅  |
