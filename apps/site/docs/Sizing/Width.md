# Width

> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class    | Native | Web |
| -------- | ------ | --- |
| w-{t}    | ✅     | ✅  |
| w-{a}    | ✅     | ✅  |
| w-full   | ✅     | ✅  |
| w-screen | ✅     | ✅  |

| w-min    | ❌     | ✅  |
| w-max    | ❌     | ✅  |
| w-fit    | ❌     | ✅  |
