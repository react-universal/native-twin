# Padding

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class   | Native | Web |
| ------- | ------ | --- |
| p-{t}   | ✅     | ✅  |
| p-{a}   | ✅     | ✅  |
| px-{t}  | ✅     | ✅  |
| px-{a}  | ✅     | ✅  |
| py-{t}  | ✅     | ✅  |
| py-{a}  | ✅     | ✅  |
| pt-{t}  | ✅     | ✅  |
| pt-{a}  | ✅     | ✅  |
| pr-{t}  | ✅     | ✅  |
| pr-{a}  | ✅     | ✅  |
| pl-{t}  | ✅     | ✅  |
| pl-{a}  | ✅     | ✅  |
| pb-{t}  | ✅     | ✅  |
| pb-{a}  | ✅     | ✅  |
| p-auto  | ❌     | ✅  |
| py-auto | ❌     | ✅  |
| px-auto | ❌     | ✅  |
| pt-auto | ❌     | ✅  |
| pr-auto | ❌     | ✅  |
| pl-auto | ❌     | ✅  |
| pb-auto | ❌     | ✅  |
