# Space Between

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class           | Native | Web |
| --------------- | ------ | --- |
| space-{t}       | ✅     | ✅  |
| space-{a}       | ✅     | ✅  |
| space-x-{t}     | ✅     | ✅  |
| space-x-{a}     | ✅     | ✅  |
| space-x-{t}     | ✅     | ✅  |
| space-y-{a}     | ✅     | ✅  |
| space-x-reverse | ❌     | ✅  |
| space-y-reverse | ❌     | ✅  |


## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/space-between) ._
