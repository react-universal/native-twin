# Text Decoration Color

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class           | Native | Web |
| --------------- | ------ | --- |
| decoration-{t}     | ✅     | ✅  |
| decoration-{a}     | ✅     | ✅  |
| decoration-inherit    | ❌     | ✅  |
| decoration-current   | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/text-decoration-color) ._
