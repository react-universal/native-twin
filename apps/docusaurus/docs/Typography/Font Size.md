# Font Size

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class    | Native | Web |
| -------- | ------ | --- |
| text-{t}    | ✅     | ✅  |
| text-{a}    | ✅     | ✅  |
| text-base   | ✅     | ✅  |
