# Flex Basis

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values 

| Class        | Native | Web |
| ------------ | ------ | --- |
| basis-{t}    | ✅     | ✅  |
| basis-{a}    | ✅     | ✅  |
| basis-auto   | ❌     |  ✅  |
