# Divide Color

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class        | Native | Web |
| ------------ | ------ | --- |
| divide-{t} | ✅     | ✅  |
| divide-{a} | ✅     | ✅  |
| divide-inherit | ❌     | ✅  |
| divide-current | ❌     | ✅  |
