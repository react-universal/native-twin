# Border Width

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class          | Native | Web |
| -------------- | ------ | --- |
| border         | ✅     | ✅  |
| border-{t}     | ✅     | ✅  |
| border-{a}     | ✅     | ✅  |
| border-x       | ✅     | ✅  |
| border-x-{t}   | ✅     | ✅  |
| border-x-{a}   | ✅     | ✅  |
| border-y       | ✅     | ✅  |
| border-y-{t}   | ✅     | ✅  |
| border-y-{a}   | ✅     | ✅  |
| border-t-{t}   | ✅     | ✅  |
| border-t-{a}   | ✅     | ✅  |
| border-b-{a}   | ✅     | ✅  |
| border-b-{t}   | ✅     | ✅  |
| border-l-{a}   | ✅     | ✅  |
| border-l-{t}   | ✅     | ✅  |
| border-r-{a}   | ✅     | ✅  |
| border-r-{t}   | ✅     | ✅  |
