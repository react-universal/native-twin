# Border Color

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class           | Native | Web |
| --------------- | ------ | --- |
| border-{t}     | ✅     | ✅  |
| border-{a}     | ✅     | ✅  |
| border-inherit  | ❌     | ✅  |
| border-current  | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/border-color) ._
