# Background Color

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class           | Native | Web |
| --------------- | ------ | --- |
| bg-{t}     | ✅     | ✅  |
| bg-{a}     | ✅     | ✅  |
| bg-inherit    | ❌     | ✅  |
| bg-current   | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/background-color) ._
