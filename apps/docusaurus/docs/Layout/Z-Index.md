# Z-Index

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class  | Native | Web |
| ------ | ------ | --- |
| z-{t}  | ✅     | ✅  |
| z-{a}  | ✅     | ✅  |
| z-auto | ❌     | ✅  |
