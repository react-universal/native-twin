# Min-Height

## Compatibility

> Nomenclature
> - -{t} Supports values from Theme
> - -{a} Support Arbitrary values  

| Class      | Native | Web |
| ---------- | ------ | --- |
| min-h-0    | ✅     | ✅  |
| min-h-{a}  | ✅     | ✅  |
| min-h-full | ✅     | ✅  |
| min-h-min  | ❌     | ✅  |
| min-h-max  | ❌     | ✅  |
| min-h-fit  | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/min-height) ._
