# Min-Width

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class      | Native | Web |
| ---------- | ------ | --- |
| min-w-0    | ✅     | ✅  |
| min-w-{a}  | ✅     | ✅  |
| min-w-full | ✅     | ✅  |
| min-w-min  | ❌     | ✅  |
| min-w-max  | ❌     | ✅  |
| min-w-fit  | ❌     | ✅  |

## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/min-width) ._
