# Max-Width

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class            | Native | Web |
| ---------------- | ------ | --- |
| max-w-0          | ✅     | ✅  |
| max-w-{a}        | ✅     | ✅  |
| max-w-{t}        | ✅     | ✅  |
| max-w-full       | ✅     | ✅  |
| max-w-screen-sm  | ✅     | ✅  |
| max-w-screen-md  | ✅     | ✅  |
| max-w-screen-lg  | ✅     | ✅  |
| max-w-screen-xl  | ✅     | ✅  |
| max-w-screen-2xl | ✅     | ✅  |
| max-w-min        | ❌     | ✅  |
| max-w-max        | ❌     | ✅  |
| max-w-fit        | ❌     | ✅  |
| max-w-prose      | ❌     | ✅  |
