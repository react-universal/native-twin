# Max-Height

## Compatibility

> Nomenclature
> - -{t} Supports values from theme
> - -{a} Support arbitrary values  

| Class        | Native | Web |
| ------------ | ------ | --- |
| max-h-0      | ✅     | ✅  |
| max-h-{a}    | ✅     | ✅  |
| max-h-{t}    | ✅     | ✅  |
| max-h-full   | ✅     | ✅  |
| max-h-screen | ✅     | ✅  |
| max-h-min    | ❌     | ✅  |
| max-h-max    | ❌     | ✅  |
| max-h-fit    | ❌     | ✅  |


## Usage

_Please refer to the [official documentation of tailwind](https://tailwindcss.com/docs/max-height) ._
